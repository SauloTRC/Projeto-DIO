Na aula 01 aprendemos um pouco sobre a história dos computadores.
A primeira maquina surgiu em 1944 ela era chamada, Mark 1, ela era utilizada para fazer calculos.
Mas a ideia do computador veio bem antes dessa primeira maquina.
Ao longo da historia tivemos uma evolução muito grande, outras maquinas foram criadas como o Colossus (1946), Eniac em (1946), ambas criadas na mesma época por pessoas diferentes e lugares diferentes.
Esses computadores eram automaticos, foi nessa época que surgiu o termo "bug".
Depois tivemos na segunda geração em (1959 a 1965) uma evolução no tamanho dos computadores, eles ficaram menores, e agora era possivel movelos de uma sala para outra.
Na terceira geraçao foram criados os circuitos integrados(1965 a 1970).
Quarta geraçao que veio de (1971) até os dias de hoje, foi marcada pela criação do personal computer, os computadores portateis, smartphones, tablets, smartwatch.
Surgimento da Web
Essa ideia de comunicação surgiu por meio do telégrafo, em 1958 foi traçado um cabo de um continente a outro.
Na decada de 1950 explodiu essas tecnologias por conta da guerra, foi criado entao a (DARPA).
Foi plantado uma ideia de que podiamos nos comunicar através de uma rede.
Depois de muito tempo surgiu a (Arpanet).
A primeira conexão foi feita em 29/10/1969, neste mesmo dia ouve tambem a primeira queda de conexão.
Em 1989, Tim Berners-lee viuu a oportunidade de unir hipertexto com TCP/IP e criar a Word Wide Web (www).
primeiro navegador chamado WordWideWeb.

O que é servers

Servidor é aqule que fornece a internet.
Existem tipos de servidores ex: arquivo, segurança, Streaming, e-mail, web.
Para efeito de conhecimento nos temos o servidor proxy, ele bloqueia alguns sites, empresas usam para restringir o a acesso a alguns sites.
Todo computador é báseado em duas partes, logica e física.
O softwares funciona como a parte lógica, que são os sistemas operacionais monitoramento, servidores e web.
Já o Hardwares funciona como a parte fisica, como o armazenamento, memoria, processadores.

HTML na Prática

(voltar)

Primeiro precisamos entender que o HTML não é uma linguagem de programaçao e sim uma linguagem de texto. O HTML consiste em três partes.
A tag "html" tem duas partes Cabeça e corpo, a cabeça que é o cabeçalho, chamamos a tag "head", para o corpo, que é o que vai aparecer para o usuario chamamos a tag "body".
A tag "i" deixa o texto em italico.
A tag "strong" deixará o texto em negrito.
A tag "input" nao tem fechamento.
A tag "img" não tem fechamento.
Algumas tags tem atributos, esses atributos são algumas propriedades que as tags podem ter.
Exemplos: A tag "strong" tem o "id" como atributo, que serve para identificar algum alemento.
Também temos como exemplo a "style" que serve para aplicar um comando ao CSS para cores.
Também como exemplo a "class" esses atributos são gênericos.
Existem atributos que são específicos de cada tag, por exemplo: "input" ele vai fazer um campo texto no "html", ele tem um atributo chamado "type=".
A tag "img" precisa do atributo "src" para copiar uma "URL" de imagens.
Para ler textos usamos a tag "h1" para titulos, temos tambem "h2,h3,h4,h5,h6" etc..
Para sub titulos podemos usar "h2".
Para paragrafos usamos "p".
Para sitações usamos a tag "blockquote".
Usamos a tag "mark" para marcar o texto.
Usamos listas ordenadas e não ordenadas no html.
Usamos as tags "ol" para lista ordenada, e dentro colocaremos a tag "li".
Para lista não ordenada usaremos a tag "u" e dentro a tag "li".
Para links utilizamos a tag "a" mais a tag "href"= exemplo "a href="http://dio.me">dio.me a"
